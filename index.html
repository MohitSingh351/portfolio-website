Parkinson's Disease Detection

In this project, I developed a machine learning model to detect Parkinson's disease from voice recordings. Utilizing both Logistic Regression and Support Vector Classification (SVC), I achieved an impressive 90% accuracy with each model. This project showcases my skills in data preprocessing, feature engineering, and model evaluation, highlighting the potential of machine learning in healthcare diagnostics.

Credit Card Fraud Detection

In this project, I developed a machine learning model to detect credit card fraud using Logistic Regression. The model achieved a 96% accuracy rate and a 96.2% AUC, demonstrating its effectiveness in identifying fraudulent transactions. This project highlights my expertise in data preprocessing, feature selection, and model evaluation, showcasing the practical application of machine learning in financial security.

Mall Customer Segmentation

In this project, I performed customer segmentation using K-means clustering on mall customer data. By analyzing customer spending habits and income levels, I identified distinct customer segments based on their purchasing behavior. This project showcases my skills in data visualization, clustering algorithms, and customer profiling, offering valuable insights for targeted marketing strategies.

Real and Fake News Detection

This project involves using machine learning to differentiate between fake and real news, achieving high accuracy with several models. This project showcases my ability to apply and evaluate various classification techniques effectively.
